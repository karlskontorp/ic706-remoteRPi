DIY remote kit for the IC-706.
Work in progress.

More info and some background is available at:
http://oz9aec.net/index.php/ic-706

The original project wqas based on BeagleBone Black. I have adapted the code to work with Raspberry Pi 3 B and
Raspian OS version Trixie.
The audio in/out is taken care of by a simple USB sound card dongle supported by RPi.
